# Relative Priority (Suggested Rollout Order)

## P0 – Credential Issuance
- Self‑issuance for employees and contractors
- Optional government ID pre‑verification for high‑risk onboarding

## P1 – Highest Value / Fastest Risk Reduction
- Account recovery and helpdesk verification
- Controlled administrator access

## P2 – Next Wave
- Privilege escalation via Identity Governance
- High‑profile user sensitive access

## P3 – Expansion
- Remote onboarding at scale
- Physical site access

> P2 / P3 order may vary based on workforce profile.
