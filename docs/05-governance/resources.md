# Microsoft Entra Verified ID demo resources

## Click-through demos

### Employee onboarding
- **Link:** [Employee onboarding](https://woodgroveemployee.azurewebsites.net/landing)
- **Description:** Interactive remote onboarding demo that walks a new employee through joining the employee portal, retrieving a verified workplace credential, and completing follow-on onboarding tasks such as equipment ordering.

### Help desk with Face Check
- **Link:** [Help desk with Face Check](https://woodgrovehelpdesk.azurewebsites.net/)
- **Description:** Help desk scenario that uses Microsoft Entra Verified ID together with Face Check to strongly verify a user before support actions are performed.

### Entra Suite demo series
- **Link:** [Entra Suite demo series](https://regale.cloud/microsoft/play/3133/enhanced-workforce-and-guest-lifecycle#/presenter/0/0)
- **Description:** A broader set of Microsoft Entra demo scenarios intended to showcase end-to-end identity, access, and verification experiences across the suite.
#### Demo Scenarios
  - IT admin Verified ID & Entitlement Management configuration: Admin-focused setup scenario covering how Verified ID and Entitlement Management can be configured to support governed access and identity-driven workflows.
  - Help desk with Face Check (demo scenario): Reference to the same help desk verification scenario, useful when you want to discuss the use case independently from the live demo URL.
  - Access to resources with Verified ID & Entitlement Management: Demonstrates how verified identity signals can be paired with governed access flows so users can request or receive access to resources in a controlled way.

### Click thru website
- **Link:** [Click thru website](https://explore.woodgrovevid.com/)
- **Description:** Interactive site for exploring Verified ID sample experiences, including obtaining and presenting credentials across sample business scenarios.
