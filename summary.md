# Executive Summary

Microsoft Entra Verified ID enables organisations to implement high-assurance identity verification for employees and external workers using privacy-preserving, decentralised credentials.

Face Check strengthens this capability by verifying that a real human is present at the moment of access. It provides strong protection against impersonation, deepfakes, and synthetic identity fraud.

# Business Benefits

Microsoft Entra Verified ID with Face Check delivers measurable security and operational benefits:

- Reduces impersonation, deepfake, and social engineering risk  
- Strengthens Zero Trust by validating both identity and real‑person presence  
- Improves security of high‑risk workflows in a privacy‑respecting way  
- Creates a consistent, user‑friendly verification experience across digital and physical scenarios
